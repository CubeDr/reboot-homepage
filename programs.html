A Variety of Class Options

Youth Class

유소년 클래스

초등 유소년 전용. 놀이처럼 시작해 운동 습관과 바른 기본기를 함께 잡습니다.

Basic Class

입문 - 기본기

라켓이 처음인 성인. 그립과 스텝부터 첫날 셔틀을 넘기는 재미까지 차근차근.

Level-Up Class

레벨업 클래스

급수 승급과 게임력 향상에 집중. 굳은 폼을 점검하고 정확도-일관성을 끌어올립니다.

Lunch Class

직장인 점심 레슨

평일 낮 시간, 쿠폰제 개인레슨으로 짧고 굵게. 바쁜 일과 사이 40분 집중 교정.

Game Class

게임 - 실전 클래스

패턴 플레이와 코트 운영을 실제 게임 상황에서. 배운 걸 시합에서 쓰게 만듭니다.

Competition Class

대회 준비반

동호인 대회 입상이 목표. 실전 전술-체력-멘탈까지 시합에서 통하게 만듭니다.

가격이 아니라, 구조가 다릅니다

90분 한 회, 충분한 시간

50분 수업으로는 교정과 실습이 끝나지 않습니다. 리부트 아카데미는 회당 90분 — 브리핑부터 복기까지 7단계 레슨 플로우가 온전히 들어가는 시간입니다.

1개의 커리큘럼 코치가 달라도, 수업은 같습니다

레슨 품질이 코치 개인의 감에 좌우되지 않도록, 전 코치가 동일한 R-시스템 표준 커리큘럼으로 지도합니다. 어느 코치에게 배워도 같은 원칙, 같은 로드맵.

R0–R2 진단하고, 기록하는 레슨 전문기관

단계별 진단으로 시작해 역량을 기록하며 성장을 관리합니다. 체육시설업 정식 등록 시설에서, 시스템으로 가르칩니다.

STEP 2

3
STEP 3
반복 체화

4
STEP 4
실전 적용

게임 상황에 바로 적용하고 즉시 피드백. 배운 게 시합에서 통하게.

Science, Not Guesswork

직선 스윙 이론

"모든 스윙은 직선이다." 회전 스윙의 오류를 역학으로 잡아내는 리부트 독자 메소드.

키네틱 체인

지면반력에서 라켓 끝까지, 힘의 전달 순서를 과학으로 설계해 파워를 만듭니다.

40분 - 15분 교정

엘리트 선수 40분, 성인 동호인 15분. 실제 케이스로 증명된 단기 교정 성과.

BWF Level 3 수준

3D 키네마틱스-끝점 궤적 최적화. 국제 엘리트 코칭 기준으로 분석하고 지도합니다.

동서양 엘리트 코치들이 공통으로 말하는 원리 — 리부트는 그걸 커리큘럼으로 만들었습니다.

진행 방식

처음 오시면
